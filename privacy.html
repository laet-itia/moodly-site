Politique de Confidentialité

Dernière mise à jour : Janvier 2026

Notre Engagement

Chez Feelink, nous prenons ta vie privée très au sérieux. Ton journal intime t'appartient, et nous nous engageons à protéger tes données personnelles.

Le texte de ton journal reste 100% sur ton appareil. On ne lit jamais ton journal et on n'y a pas accès.

Données Collectées

Feelink collecte uniquement les données nécessaires au fonctionnement de l'app :

  • Texte du journal : stocké uniquement sur ton appareil, jamais envoyé à nos serveurs
  • Métadonnées : date, humeur, tags et stickers (synchronisés via notre serveur sécurisé)
  • Synchronisation : via iCloud et notre serveur cloud pour retrouver tes données sur tous tes appareils
  • Préférences : thème choisi, paramètres de notification, code PIN (stocké localement)
  • Achats : gérés par Apple, nous n'avons pas accès à tes informations de paiement

Stockage des Données

Tes données sont stockées :

  • Localement sur ton appareil (tout, y compris le texte de ton journal)
  • Sur iCloud (si activé) pour la sauvegarde
  • Sur notre serveur sécurisé : uniquement les métadonnées (date, humeur, tags, stickers) pour la synchronisation multi-appareils

Important : Le contenu texte de ton journal n'est JAMAIS envoyé à nos serveurs. Il reste 100% privé sur ton appareil. Nous utilisons une authentification anonyme : aucun email ni mot de passe n'est requis.

Utilisation des Données

Tes données sont utilisées uniquement pour :

  • Afficher tes entrées de journal
  • Calculer tes statistiques d'humeur
  • Synchroniser entre tes appareils (via iCloud)
  • Envoyer des rappels locaux (si activés)

Partage des Données

Nous ne vendons, ne louons et ne partageons jamais tes données personnelles avec des tiers.

Les seules exceptions :

  • Apple : pour la synchronisation iCloud et les achats in-app
  • Obligations légales : si la loi nous y oblige

Sécurité

Nous utilisons les mesures de sécurité suivantes :

  • Chiffrement des données par Apple (iCloud)
  • Protection par Face ID / Touch ID / Code PIN
  • Aucune transmission de données vers des serveurs tiers

Tes Droits

Tu peux à tout moment :

  • Supprimer tes données depuis l'app (Paramètres › Réinitialiser)
  • Désactiver la synchronisation iCloud

Mineurs

Feelink peut être utilisée par des personnes de tout âge. Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants de moins de 13 ans sans le consentement parental.

Modifications

Nous pouvons mettre à jour cette politique de confidentialité. En cas de changement significatif, nous t'en informerons via l'app.
